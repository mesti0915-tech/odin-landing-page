#  Services Landing Page

A clean and professional landing page designed for a home maintenance and repair company.

This project demonstrates how to build a complete website layout using HTML and CSS,
following a modern and easy-to-read design structure.

## Features
- Clear hero section with call-to-action
- Service overview section
- Customer testimonial section
- Call-to-action banner
- Simple and clean footer

## Technologies Used
- HTML5
- CSS3 (Flexbox)
